This directory contains fault-tolerant circuits from different papers.

* `FAO-FTStatePrep`: circuits from https://arxiv.org/abs/2508.14200
* `RL-FTStatePrep`: circuits from https://arxiv.org/abs/2402.17761
* `SAT-FTStatePrep`: circuits from https://arxiv.org/abs/2408.11894
* `Steane-FTStatePrep`: circuits from https://arxiv.org/pdf/2601.13313

Furthermore, the `miscellaneous` folder contains various state preparations circuits from papers where only a single code was used. 
* `carbon_code`: circuit from https://arxiv.org/pdf/2404.02280
* `smallest_interesting`: circuit from https://arxiv.org/pdf/2309.09893
* `32_20_4`: circuit from private conversation with Ben Criger
